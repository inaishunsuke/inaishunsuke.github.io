JOINT 　（関数）


ジョイント型座標から角度を抽出します。


JOINT(<軸番号>,<ジョイント型>)


<ジョイント型>で指定したジョイント型座標から、<軸番号>で指定した一つの軸の関節角度を抽出します。


LETJ


DEFJNT lj1, lj2
DEFSNG lf1, lf2
DIM li1 As Integer
lf1 = JOINT(1, lj1)
'lj1の1軸の関節角度をlf1に代入します。
lf2 = JOINT(li1, lj2)
'lj2のli1の値の軸の関節角度をlf2に代入します。

6軸
lf1 = JOINT(l, lj1 + (10, 10, 0, 0, 0, 0) )
'lj1 + (10, 10, 0, 0, 0, 0)の1軸の関節角度をlf1
'に代入します。

4軸
lf1 = JOINT(l, lj1 + (10, 10, 0, 0) )
'lj1 + (10, 10, 0, 0)の1軸の関節角度をlf1に代入
'します。
