DIST 　（関数） 　【SLIM準拠】


2点間の距離を返します。


DIST(<ポジション型1>,<ポジション型2>)


<ポジション型1>と<ポジション型2>との距離を得ます。


DEFSNG lf1, lf2
DEFPOS lp1, lp2, lp3

6軸
lf1 = DIST((350, 0, 450, 0, 0, 180), lp1)
'(350, 0, 450, 0, 0, 180)とlp1間の距離をlf1に代
'入します。

4軸
lf1 = DIST((350, 0, 300, 45), lp1)
'(350, 0, 300, 45)とlp1間の距離をlf1に代入しま
'す。

lf2 = DIST(lp2, lp3)
'lp2とlp3間の距離をlf2に代入します。
