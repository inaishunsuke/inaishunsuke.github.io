TOOL 　（ステートメント）


ツール座標系を宣言します。


TOOL <ツール座標系番号>,<ポジション型>


<ポジション型>で示されるポジションを、<ツール座標系番号>で示されるツール座標系として宣言します。
ツール座標系番号には、1～63が指定できます。


CHANGETOOL、TOOLPOS


このコマンドにより変更される値は、電源が入っている限り値を保持します。電源が切れても値を保持したい場合は、システムパラメータの保存（操作ガイドP5-181参照）を行ってください。


6軸
DIM lp1 As position
TOOL 1, lp1
'lp1を、ツール座標系番号1のツール座標系として宣
'言します。
TOOL 2, (100, 100, 50, 0, 90, 0 )

4軸
DIM lp1 As position
TOOL 1, lp1
'lp1を、ツール座標系番号1のツール座標系として宣
'言します。
TOOL 2, (100, 100, 50, 0 )
