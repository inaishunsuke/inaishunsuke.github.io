POSX 　（関数） 　【SLIM準拠】


X成分を抽出します。


POSX({<ポジション型>|<ベクトル型>})


ポジション型またはベクトル型の座標から、X成分を抽出します。


POSY、POSZ


DIM lf1 As Single
DIM lp1 As Position
lf1 = POSX(lp1)
'lp1のX成分をlf1に代入します。

6軸
lf1 = POSX(lp1 + (100, 100, 100, 0, 0, 0) )
'lp1 + (100, 100, 100, 0. 0, 0)のX成分をlf1に代
'入します。

4軸
lf2 = POSX(lp1 + (100, 100, 100, 0) )
'lp1 + (100, 100, 100, 0)のX成分をlf1に代入しま
'す。
