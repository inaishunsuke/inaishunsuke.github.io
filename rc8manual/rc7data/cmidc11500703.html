PVEC 　（関数）


位置ベクトルを抽出します。


PVEC({<同次変換型>|<ポジション型>})


同次変換型座標または、ポジション型座標から、位置ベクトルを抽出します。


AVEC、OVEC、RVEC


DIM lt1 As Trans
DIM lv1 As Vector
DIM lp1 As Position
lv1 = PVEC(lt1)
'lt1の位置ベクトルをlv1に代入します。

6軸
lv1 = PVEC(lp1 + (100, 200, 0, 0, 0, 0) )
'lp1 + (100, 200, 0, 0, 0, 0)の位置ベクトルをlv1
'に代入します。

4軸
lv1 = PVEC(lp1 + (100, 200, 0, 0) )
'lp1 + (100, 200, 0, 0)の位置ベクトルをlv1に
'代入します。
