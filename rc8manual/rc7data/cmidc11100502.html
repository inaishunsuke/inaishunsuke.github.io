IF...THEN...ELSE 　（ステートメント） 　【SLIM準拠】


論理式の条件判断を行ないます。


IF <条件式> THEN {<文>|<ラベル名>} [ELSE {<文>|<ラベル名>}]


<条件式>の条件によってプログラムの実行を制御します。
<条件式>が真(0以外)ならばTHEN以降を実行し、<条件式>が偽(0)ならばELSE以降が実行されます。


IF...END IF


IF i1 = 0 THEN STOP ELSE GOSUB *samp1
'ilが0の場合プログラムの実行を終了し、ilがその
'他の場合、ラベル名*samp1のサブルーチンを
'呼び出します。
il = il + 1
'ilを加算します。
END
'プログラムの終わりを定義します。
*samp1:
'サブルーチンのラベルを定義します。
i0 = 0
'i0に0を代入します。
RETURN
'元のプログラムに戻ります。
