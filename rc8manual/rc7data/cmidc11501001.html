FIG 　（関数）


形態を抽出します。


FIG({<ポジション型>|<同次変換型>})


ポジション型、同次変換型から形態を抽出します。


CURFIG、LETF、ロボット形態(付録2)


DIM li1 As Integer
DIM lp1 As position
li1 = FIG(lp1)
'lp1の形態をli1に代入します。

6軸
li1 = FIG(lp1 + (0, 0, 100, 0, 0, 0) )
'lp1 + (0, 0, 100, 0, 0, 0)の形態をli1に
'代入します。

4軸
li1 = FIG(lp1 + (0, 0, 100, 0) )
'lp1 + (0, 0, 100, 0)の形態をli1に代入し
'ます。
